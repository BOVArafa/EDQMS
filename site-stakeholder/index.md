## How to use this site
- create tree tracks for consuming the content of the site:
  - Managers: For those who need just to understand what the project is all about so they can participate in followup meetings to make decisions.
    - These users must, at least, read the intro subpages of every page and than go to the Q&A
  - Owners: For those responsible for keeping the project on track.
    - These users need to have a good overall understand about the project and understand how the roadmap is setup and how to follow it up.
  - Actors: For those who needs to take actions to develop the project
    - These users must know the project in details to be able to execute the tasks in the roadmap

## Q&A
- Add questions, at least 10, so that stakeholders can have a overall understanding of the project without the need of reading everything. Examples:
  - Why this project begin and when will I know it ended?
  - How can I convince my boss to invest in this project?
  - Why does it feels like this belongs to a Quality Department project?
- Where are we with the project now?
